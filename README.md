# iDent
Ident Video for publicly identifying a person, company, venue, project or video game

### What is Ident Video?
Everyone can generate a short 10 second punchy video that identifies something \
using their own game engine or content creation pipeline to make it unique.

### How do I use Ident Video?
Simply play back the video in a chat or presentation as yourself to reference it. \
Keep it safe, you're going to want to use the same pre-edited clips multiple times.

### What if I can't use Ident Video?
It is a digital alternative to what is otherwise known as a profile picture or pfp. \
You could use a screenshot, audio clip or video badge if your device or medium permits it.
